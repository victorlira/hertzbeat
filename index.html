Open Source Real-time Monitoring System

Quickstart

Convenient Used

docker run -d -p 1157:1157 apache/hertzbeat
Monitoring-alarm-notify all in one. Support monitoring web service, database, cache, os, web-server, middleware, bigdata, cloud-native, network etc.
Easy to use and agentless, full web-based operations with just a click of a mouse.
Security is the most important, encrypt the whole secret data.

High Performance and Custom

Makes protocols such as Http, Jmx, Ssh, Snmp, Jdbc configurable, you can collect any metrics by simply configuring the yml online.
High performance, supports horizontal expansion of multi-collector clusters, multi-isolated network monitoring and cloud-edge collaboration.
Flexible threshold rules and timely notifications delivered via discord slack email webhook more.

Opensource Friendly

Apache HertzBeat (incubating)'s stand-alone and cluster version code are all open source, unlimited and anyone who are interested in it are very welcome to contribute. No matter how small the contribution is, whether it is a document code or a typo, we respect everyone and learn and grow together.
Look forward our code to be deployed on thousands of machines worldwide.Github Repo
